**Project Title:**

Plant Leaf Disease Detection & AI Chatbot for Fertilizer Suggestion

**Description:**

This project uses Machine Learning and Computer Vision to detect plant leaf diseases from images and provides fertilizer recommendations using an AI-based chatbot. It helps farmers take quick and accurate decisions to improve crop health and yield.

**Technologies Used:**

- Python  
- Flask  
- PyTorch (`torch`, `torchvision`)  
- OpenCV  
- NumPy, Pandas  
- Pillow (Image Processing)

**Features:**

🌿 Upload leaf image → Detect disease

🤖 AI chatbot → Suggest fertilizer

📊 Accurate prediction system

**How to Run:**

Install requirements

Run app.py
